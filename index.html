... About Relocation costs.
The first 20km are free.
If the location where you want photos/videos is over 20km, then there will be travel costs. This is €0.40 per Km. During a photo/video shoot, Can be chosen to go to multiple locations.
Some packs include multiple locations. So if 1 location is included this is the location we discuss together
if 2 or more locations are included, we agree on several locations to take different photos/videos
(this must be within 10 km of the first agreed location.) if this is further than 10km, travel costs will be added again - (€0.40)
With the video - max&ultra packs, travel costs will only be charged if this is further than 30km.
